# CodeAlpha Image Gallery

This is a simple Image Gallery project created using HTML, CSS, and JavaScript.

## Features
- Display multiple images in a gallery layout
- Responsive design
- Smooth hover effects
- Easy to add or remove images

## How to Run
1. Download or clone the repository
2. Open `image-gallery/index.html` in a browser
