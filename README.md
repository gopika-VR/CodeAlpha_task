# CodeAlpha Projects

This repository contains multiple web projects developed as part of the CodeAlpha internship tasks using HTML, CSS, and JavaScript.

## Projects Included

### 1. Calculator
A simple calculator application for performing basic arithmetic operations.

**Features:**
- Addition
- Subtraction
- Multiplication
- Division
- Clean and simple UI

**Run Instructions:**
Open `calculator/index.html` in a web browser.

---

### 2. Image Gallery
A responsive image gallery to display multiple images with a clean layout.

**Features:**
- Responsive grid layout
- Hover effects on images
- Easy image add/remove
- User-friendly design

**Run Instructions:**
Open `image-gallery/index.html` in a web browser.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Use
1. Clone or download this repository
2. Open any project folder
3. Run the `index.html` file in a browser
